RECONNAISSANCE


1)eyewitness

https://www.christophertruncer.com/eyewitness-usage-guide/


2)awsbucket dump

This is a security tool to find files in AWS S3 buckets.This is used for

4)nmap

discover hosts,open ports and detecting security risks on a computer network.
https://github.com/nmap/nmap
https://nmap.org/download.html


5)dnsrecon

DATA EXFILTRATION

It is a unauthorized copying,tranfer of data from a computer or server.


1)cloakify factory

It is a tool to transform  any type of file into a list of harmless and even useless 
looking strings. This is to hide a data file in plain sight and used to transfer it
over any network without any alerts.
https://github.com/TryCatchHCF/Cloakify


2)det

It is a tool to exfiltarating data using either one or several channels at the same time.
It is used to test the data leakage prevention solutions configuration.
https://github.com/sensepost/DET


3)dns exfiltration

It is a way to exchange data between two computers without any direct connection,
the data will exchange through DNS protocol. 
https://github.com/Arno0x/DNSExfiltrator


4)pyexfil

The purpose of this is to set as many exfiltration,and now also communication,techniques
that can be used by various malware around to bypass various detection and migration tools
and techniques.
https://github.com/ytisf/PyExfil


5)egress-assess

This is a tool to test egress data detection capabilities.It is used to test if there
is a security control that will prevent sending data to to remote location.
https://github.com/ChrisTruncer/Egress-Assess
https://www.christophertruncer.com/egress-assess-testing-egress-data-detection-capabilities/
https://www.christophertruncer.com/egress-assess-action-via-powershell/


6)powershell rat

It is a powershell script tool that has been made to help a pentester to backdoor windows.
It tracks user activity and sends information to us as an e-mail attachment.
https://github.com/Viralmaniar/Powershell-RAT
